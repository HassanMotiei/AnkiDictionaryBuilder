# Anki Dictionary Builder

A Python application for converting MDict dictionaries (MDX/MDD) into high-quality Anki flashcards.

## Features

- Read MDX dictionaries
- Parse HTML entries
- Store data in SQLite
- Extract audio from MDD
- Generate TSV files for Anki
- Future support for Oxford, Longman, Cambridge and Collins dictionaries

## Project Structure

- dictionaries/
- src/
- database/
- input/
- output/
- media/
